href="https://github.com/kmbt/hydrogen-renamer">script</a> will help you rename the Gantry 5 Hydrogen theme as a starting point for further customizations.</p> <p>Recently, I've been trying out the <a href="http://gantry.org">Gantry 5</a> - a theme framework for Wordpress. One way to get started with theme development using Gantry 5 is to take the default theme, called Hydrogen, rename it and customize it to one's needs.</p> <p>The whole renaming process is described <a href="http://docs.gantry.org/gantry5/theme-development/change-theme-name">here</a>. It takes few steps, which involve unzipping the theme, changing names of files and directories and doing a <em>find and replace</em> for the word "hydrogen" written in few combinations of uppercase and lowercase letters. </p> <p>I thought that the whole process could be easily automated with a simple Python script that would repack the theme on the fly while doing all the necessary string substitutions in file contents as well as in the paths. </p> <p>The script is available <a href="https://github.com/kmbt/hydrogen-renamer">here on Github</a>.
